Rust and ash voxel engine expiriment!
I am trying to make a minimal voxel engine in Rust using the Vulkan Rust bindings called ash. The vision is to make every step as 
performant as possible to be able to allow for as much cool stuff in game as possible. Part of the idea and purpose is to improve 
on other games within the same genre. 

At this point in time this is in reality only a small experimental learning focused project though, but this could change if I see 
enough short term success!
